# House Price Prediction using Linear Regression

## Description

This project aims to predict the price of a house based on its area (in square feet). A simple Linear Regression model is used to understand the relationship between house size and price.

## Dataset

* Source: Sample dataset (manually created)
* Description of data:

  * Area: Size of the house in square feet
  * Price: Cost of the house

## Steps Performed

1. Data Cleaning

   * Used structured numerical data
   * No missing values in dataset

2. Exploratory Data Analysis

   * Observed relationship between area and price

3. Visualization

   * (Optional) Scatter plot to visualize linear relationship

4. Model Building

   * Used Linear Regression model
   * Trained model on sample dataset
   * Predicted house price based on user input

## Results

* Model predicts house prices based on area
* Shows a linear relationship between input and output

## Tools Used

* Python
* NumPy
* Scikit-learn

## Conclusion

This project demonstrates a basic regression model for predicting continuous values. It is simple and effective for understanding machine learning fundamentals.
