# Water Quality Analysis: Predicting Water Pollution

## Description

This project aims to predict whether water is clean or polluted based on key water quality parameters such as pH, temperature, and turbidity. The goal is to build a simple machine learning model that can help in identifying potential pollution in water bodies.

## Dataset

* Source: Sample dataset (manually created)
* Description of data:

  * pH: Acidity or alkalinity of water
  * Temperature: Water temperature in degrees Celsius
  * Turbidity: Clarity of water
  * Pollution: Target variable (0 = Clean, 1 = Polluted)

## Steps Performed

1. Data Cleaning

   * Checked for missing values
   * Ensured correct data types

2. Exploratory Data Analysis

   * Observed distribution of water parameters
   * Compared clean vs polluted samples

3. Visualization

   * Basic analysis of feature values (optional plots can be added)

4. Model Building

   * Used Decision Tree Classifier
   * Split dataset into training and testing sets
   * Trained model using Scikit-learn

## Results

* The model successfully classifies water as clean or polluted
* Achieved good accuracy on test data (based on small dataset)

## Tools Used

* Python
* NumPy
* Pandas
* Scikit-learn

## Conclusion

The project demonstrates how machine learning can be used to analyze water quality and detect pollution. With a larger and real-world dataset, the model can be improved further for practical applications.
